FAQ and Help

This document is to aid in understanding what the dashboard is displaying and how best to use the controls provided.

What Data is This Using?

This dashboard uses data collected from Google analytics and Google Tag Manager. The dashboard updates daily, grabbing data for the previous day. This means that the dashboard will always display data that is at least one day old. This is done to ensure we do not provide an incomplete picture about a given day.

What Do These Plots Show?

These plots display conversions and allow the user to specify various views in order to facilitate discovery and understanding of the new mass.gov.


Visitor Success - Success by Group

This plot shows the absolute number or percentage of conversions each of the groups on the x-axis are tied to. We also provide a pareto line when "Display Unit" is set to "Percent."


Visitor Success - Success Over Time

This plot shows the conversion rate or absolute number of conversions over time according to the groups found on the x-axis.


Success Rate - Sessions on Top Topics

The sessions on top topics plot displays the volume of sessions each of the top topics on mass.gov received.


Success Rate - Conversions on Top Topics

Conversions on top topics displays the % of conversions that occurred on a top topic or the absolute number of conversions that occurred on that topic.


Success Rate - Conversion Rate for Top Topics

This plot shows the conversion rate (sum of Sessions w. Conversion / sum of Sessions), it is the logical combination of the Sessions on Top Topics plot and the Conversions on Top Topics plot.


What is Considered a Session

A session is defined as a group of interactions one user takes without a given time frame of inactivity on our website. We set that time frame to 30 minutes. Meaning whatever a user does on our website (e.g. browses pages, downloads resources) before they leave or go inactive for 30 minutes or more equals one session.

What is Considered a Conversion?

A conversion is an event which can only occurr on the final page of a users session. A conversion is an action which indicates a user completed a desired action during their session either dumping them into a funnel or providing them with information or a document that will help them further complete their desired action. The following actions are considered conversions: any click to a quick action link, contact link clicks, document downloads, and callout link clicks.
